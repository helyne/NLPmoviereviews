# Install

Clone the project and install it:

```bash
git clone git@github.com:helyne/NLPmoviereviews.git
cd NLPmoviereviews
pip install -r requirements.txt
make clean install test                # install and test
```
Functionnal test with a script:

```bash
cd
mkdir tmp
cd tmp
NLPmoviereviews-run
```
